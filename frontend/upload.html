Lead-Import aus CSV

Importieren Sie Leads aus einer CSV-Datei. Jede Zeile wird als ein Lead importiert.

oder per Drag & Drop

Nur CSV-Dateien (mehrere Dateien möglich)

Erwartetes CSV-Format:

Format 1: Vertikales Format (EMPFOHLEN - einfach zu erstellen)

  • Spalte A: Kategorie, Spalte B: Wert
  • Erste Zeile: Kategorie,Wert
  • Jeder Lead durch eine leere Zeile getrennt
  • Kategorien: Anbieter_Name, Postleitzahl, Ort, Grundstücksfläche, Wohnfläche, Telefonnummer, Features

Format 2: Horizontales Format

  • Spalten: Anbieter_Name, Postleitzahl, Ort, Grundstücksfläche, Wohnfläche, Telefonnummer, Features
  • Features können kommagetrennt sein (z.B. "Keller, Balkon, Garten")
  • Erste Zeile muss die Spaltenüberschriften enthalten
  • Hinweis: Leads werden ohne Makler-Zuordnung importiert (Status: Unqualifiziert)
